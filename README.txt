Contains info about the files in this project

directory SIR: contains SIR models
reinfect_SIR.m: non-delay solver for SIR model with kappa parameter
reinfect_SIR_test.m: Runs reinfect_SIR.m to make figures from the book
death_SIR.m: SIR model with birth, natural death, and disease death
death_SIR_test.m: Runs death_SIR.m and plots
delay_SIR.m: Delay model, including death and vaccination
delay_SIR_test.m: Runs the H1N1 Morocco parameters to verify delay_SIR.m.
